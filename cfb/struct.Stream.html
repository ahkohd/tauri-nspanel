source

pub fn set_len(&mut self, size: u64) -> Result<()>

Truncates or extends the stream, updating the size of this stream to become size.

If size is less than the stream’s current size, then the stream will be shrunk. If it is greater than the stream’s current size, then the stream will be padded with zero bytes.

Does not change the current read/write position within the stream, unless the stream is truncated to before the current position, in which case the position becomes the new end of the stream.
