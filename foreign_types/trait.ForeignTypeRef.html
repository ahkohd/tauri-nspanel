Expand description

A trait implemented by types which reference borrowed foreign types.

§Safety

Implementations of ForeignTypeRef must guarantee the following:

  • Self::from_ptr(x).as_ptr() == x
  • Self::from_mut_ptr(x).as_ptr() == x
